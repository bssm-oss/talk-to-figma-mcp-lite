# Support

This project is maintained as a frozen prototype and local fallback.

## Where To Start

- Setup help: `docs/setup-local-dev.md`
- Lite workflow: `docs/lite-tool-flow.md`
- Original tool list: `docs/original-mcp-tools.md`
- Project status: `docs/project-status.md`
- License and attribution: `docs/lineage-license.md`

## What Maintainers Can Help With

Reasonable support topics:

- local setup problems
- MCP config examples
- Figma plugin linking problems
- WebSocket relay connection issues
- Lite tool behavior questions
- documentation fixes
- reproducible bugs in the current local workflow

## What Is Out Of Scope

This repo is not the right support channel for:

- official Figma MCP account, billing, OAuth, or seat issues
- hosted production deployments
- broad feature requests that would turn this into a new product
- private Figma file debugging without a minimal reproduction
- `DESIGN.md` orchestration work outside this repo

## Bug Reports

Please include:

- operating system
- Bun version
- MCP client name and version
- exact command or tool call
- whether the relay is running
- whether the Figma plugin is connected and joined to the same channel
- relevant stderr logs with secrets removed

## Security Reports

Use `SECURITY.md` for vulnerability reports.
